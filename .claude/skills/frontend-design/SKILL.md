# Frontend Design Skill

## Context
SellerSaathi targets Indian SMB sellers. The UI must be professional, trustworthy, and accessible.

## Tools
- Next.js 15 App Router
- Tailwind CSS v4
- shadcn/ui

## Guidelines
1. Clean, modern, glassmorphism-inspired components.
2. Clear indicators for Credit usage (Free vs Pro).
3. Responsive tables for Listings/POAs.
4. Support for RTL or multi-language fonts if required for Indian languages.
